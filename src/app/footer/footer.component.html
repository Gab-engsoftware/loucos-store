Sobre

Esta é a LoucosStore uma loja de produtos tecnológicos fictícia. Este projeto teve como objetivo elaborar uma loja completa de produtos de computação, com barra de pesquisa, links de navegação, página de contato e outras funcionalidades que é possível testar navegando no site, outro objetivo deste projeto foi enraizar os conhecimentos em Angular. Toda a loja foi desenvolvida com Angular e TypeScript. Navegue no projeto e não esqueça de deixar sua opinião de melhoria na página de contatos!
